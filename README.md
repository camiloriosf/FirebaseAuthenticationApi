## How to use

### Setup

Download the example [or clone the repo](https://github.com/camiloriosf/FirebaseAuthenticationApi.git):

Install Firebase Tools

```bash
npm install -g firebase-tools
```

Initialize project

```bash
firebase init
```

Deploy to Firebase

```bash
firebase deploy
```
